About me

Selamat Datang, saya Pebrian, saya lulusan dari SMK Al Falah Dago dari jurusan Rekayasa Perangkat Lunak. Memiliki pengalaman membuat website Sistem Informasi PRAKERIN (SIP) yang dibuat menggunakan framework laravel dan bootstrap. Saya juga berpengalaman bekerja paruh waktu di StartUp akuakultur.

My Project

Sistem Informasi PRAKERIN
Sistem Informasi Prakerin adalah sebuah sistem informasi berbasis website yang memudahkan sekolah untuk menginput data PRAKERIN seperti data siswa, perusahaan yang bekerja sama dengan sekolah, dan kegiatan siswa saat melaksanakan PRAKERIN. Project ini dibuat menggunakan framework Laravel dan Bootstrap. Project ini saya kerjakan bersama team. Website ini pertama kali project yang cukup besar yang saya buat bersama team.
